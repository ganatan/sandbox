# springboot-starter


---

## Configuration 


```xml
<artifactId>springboot-starter</artifactId>
<version>1.0.0</version>
<name>springboot-starter</name>
<description>Demo project for Spring Boot Starter</description>
<url/>
<licenses/>
```

## Build


```bash
mvn clean install
```


```text
target/springboot-starter-1.0.0.jar
```
